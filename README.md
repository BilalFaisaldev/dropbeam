# ⚡ DropBeam WiFi

DropBeam WiFi is a modern, premium, and privacy-focused local Web application designed for instant, zero-configuration file and text sharing. Inspired by the sleek aesthetics of Linear, Vercel, and Apple's Human Interface Guidelines, it provides a seamless, distraction-free environment to share assets across devices instantly.

Built with a dark-themed, ultra-fast UI, DropBeam WiFi enables users to drop files, sync clipboards, and communicate in real-time within a shared digital room—**all without creating an account.**

---

## ✨ Features

- **🌐 Active Rooms:** Share instantly inside public or private virtual rooms with a unique web link or QR code.
- **📋 Live Shared Clipboard:** Paste text or code snippets that instantly sync across all active devices in the room.
- **📁 Drag & Drop File Sharing:** Share multiple files (up to 100MB per file) with a clean card-style preview interface.
- **💥 Self-Destruct Mode:** Toggle files to automatically delete immediately after their first successful download.
- **⏱ Auto-Expiry:** Files and text snippets automatically expire and purge after 15 minutes to guarantee data privacy.
- **📱 Mobile-First & Responsive:** Perfectly optimized for seamless sharing between desktop browsers and smartphones.
- **🎨 Premium UI/UX:** High-quality micro-interactions, soft shadows, rounded corners, and a clean grid system.

---

## 🛠 Tech Stack

- **Frontend:** HTML5, Tailwind CSS, JavaScript (ES6+)
- **Animations & Icons:** Smooth CSS transitions & premium iconography
- **Hosting / Backend:** Node.js / Render (Optimized for quick temporary in-memory/disk storage)

---

## 🚀 How It Works

1. **Open:** Open the application web link on any device.
2. **Connect:** Scan the generated QR code or share the link with another device to join the same room.
3. **Drop & Share:** Paste your text into the Shared Clipboard or drag-and-drop files into the drop zone.
4. **Auto-Purge:** Your shared assets are automatically wiped from existence after the timer expires or upon download.

---

## 🔒 Privacy & Trust First

- **No Account Required:** Start sharing immediately without sharing your email or personal information.
- **Zero Persistent Tracking:** Rooms operate on temporary identifiers.
- **Auto-Delete Architecture:** Files are stored temporarily and are completely cleared automatically.
